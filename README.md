# 📚 Proyecto de Programación Orientada a Objetos

## ✨ Descripción
Este proyecto fue desarrollado como parte de la asignatura de Programación Orientada a Objetos (POO). El objetivo principal es aplicar conceptos fundamentales como:

- Clases y objetos
- Encapsulamiento
- Herencia
- Polimorfismo
- Abstracción
- Modularidad y reutilización de código

Además de aprender POO, este trabajo también representa horas de debugging, café y desarrollo de personaje.

---

## 🛠️ Tecnologías utilizadas

- Lenguaje: `Java`
- IDE: `IntelliJ IDEA / Eclipse / NetBeans`
- Paradigma: `Programación Orientada a Objetos`
